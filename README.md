# AI-Snake-Game

AI Snake Game written in python. Use curses module, Windows users 
should install it first:

<http://www.lfd.uci.edu/~gohlke/pythonlibs/#curses>


## How to run

	git clone https://github.com/tusharchoudhary0003/AI-Snake-Game.git
	cd snake-ai
	python snake.py
